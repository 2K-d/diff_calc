# Diff-Calc
This is a CLI program to compute Etterna diff on Quaver map.
## Usage
```shell
diff_calc --input <*.qua> [--rate <rate>]
```
- -i, --input FILE  
Define the FILE, a quaver map, to compute diff on.
- -r, --rate NUM  
Define rate NUM that the diff should represent, default is 1.0.
